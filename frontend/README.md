# Savor Bistro Expo

This is an Expo Go version of the Savor Bistro app.

## Run

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go.

## Code structure

- `App.tsx` controls screen navigation and shared state.
- `src/screens/` contains each separate screen.
- `src/components/` contains reusable UI components.
- `src/modals/` contains popup/modal screens.
- `src/data/` contains menu data and admin emails.
- `src/types/` contains TypeScript types.
- `src/styles/` contains shared styles.
